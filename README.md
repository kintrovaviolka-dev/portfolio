# 🩺 Viola Kintrová – Portfolio 💻

### Medical Student @ MUNI | Security-First Full-Stack Developer

Tento repozitář obsahuje mé osobní portfolio, které vizuálně i technicky propojuje dva světy: medicínu a moderní webový vývoj. Postaveno na čistém kódu, bez těžkopádných frameworků.

## 🚀 Technické Highlighty
- **DNA Helix Animation:** Procedurálně generovaná animace na `canvas` (čistý JavaScript), která simuluje 3D rotaci pomocí hloubky a opacity.
- **Security-First Focus:** V mých projektech (jako EuRegMe E-shop) kladu důraz na SQL injection prevention (prepared statements) a bezpečnou manipulaci s daty (Web Crypto API).
- **Moderní CSS:** Využití `@property` pro plynulé animace gradientů, CSS proměnné a responzivní design s důrazem na přístupnost (`prefers-reduced-motion`).
- **PWA Ready:** Optimalizováno pro rychlé načítání a offline-first přístup.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (BEM-ish), Vanilla JavaScript (ES6+).
- **Backend:** PHP (mysqli), MySQL, Google Apps Script.
- **Nástroje:** Cursor, Windsurf, Git.
